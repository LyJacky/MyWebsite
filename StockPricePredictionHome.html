Stock Application

I served as the project leader for the Stock Application project, overseeing which features needed development and managing my team accordingly. I personally implemented several key features, including generating stock graphs from JSON data provided by a stock API. The application was built using Vue.js for the front end, with Node.js handling the back end and connecting to a MongoDB database for user information storage.

Functionalities

The application allowed users to start with a balance of $10,000 in mock money. Users could buy and sell stocks, with their total balance adjusted based on their trades. Additionally, the application featured a stock price prediction tool.

Predict Future Price Feature

To predict the price of a stock for the following day, we employed an LSTM (Long Short-Term Memory) model. LSTMs are particularly effective at capturing long-term and short-term patterns in sequential data, making them ideal for stock price predictions. We trained a separate LSTM model for each stock to account for different trading patterns. Once trained, each model was used directly for future predictions without needing retraining, ensuring efficient and accurate price forecasts.
